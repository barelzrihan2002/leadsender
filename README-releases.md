# LeadSender

Official releases for LeadSender - WhatsApp Campaign Manager

## Download

Download the latest version from [Releases](https://github.com/barelzrihan2002/leadsender-releases/releases)

## Installation

1. Download `LeadSender-Setup-x.x.x.exe`
2. Run the installer
3. Follow the setup wizard
4. Enter your license key when prompted

## Auto-Updates

The application automatically checks for updates and will notify you when a new version is available.
